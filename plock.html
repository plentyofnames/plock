Step panel & parameter editing
Opening: ⌘+click a step to open its parameter panel. All SRC, SMPL, FLTR, AMP, and LFO parameters are shown with their current value — either a plock (per-step override) or the kit default.
Sliders: Drag left/right to set a plock value. The slider range always extends one tick past the maximum — dragging all the way to the right clears the plock and restores the kit default (shown as “TRK” for note, velocity, and sound lock parameters).
Sound locks: The SND slider only shows pool sounds that are actually saved on the AR. Slide to “TRK” to remove the sound lock and use the kit’s track sound. Sound locks override the machine type and all kit defaults for that step.
LFO destination: The DST slider scrolls through all valid destinations (OFF, synth params with machine-specific names, sample, filter, and amp params). When a step has a sound lock, the synth param names reflect the locked sound’s machine.
Trig flags: Toggle accent, mute, swing, slide, and retrig flags directly in the step panel. Retrig parameters (count, rate, length, velocity curve) appear when retrig is enabled.
Track defaults: Click a track name (BD, SD, …) to edit the default note, velocity, note length, track speed, and trig probability that apply to all steps without per-step overrides.
Audio preview coverage
The built-in preview is a rough rhythmic sketch, not a hardware-accurate emulator. Voices are hand-rolled 909-flavour synthesis picked by machine type — they do not read the kit’s filter, envelope, LFO, or SYN parameter bytes.
Audible: machine type (per track or sound-locked), AMP volume & pan (kit default / sound-lock default), trig enable / mute / accent, retrig (count, rate, length, ∞), swing flag, micro-timing.
Ignored: all parameter locks (plocks), parameter slides, slide flag, note value & length, filter / envelope / LFO settings, SYN parameters beyond machine selection, delay / reverb / distortion / compressor sends and FX, sample playback (SMP is silent).
Plock away freely — the hardware will play what you see, the preview just won’t reflect those tweaks.
Connects to a MIDI port whose name contains “rytm” (case-insensitive). Requests the current workbuffer pattern and kit via SysEx. FW 1.70 binary format (compatible with 1.72).
